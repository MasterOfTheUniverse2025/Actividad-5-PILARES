Sistemas Web

Blog Web

Un blog web es un sitio en línea donde una persona o grupo publica contenido regularmente, como artículos, noticias, opiniones o experiencias personales. Suelen estar organizados de forma cronológica, con las entradas más recientes al principio. Los blogs permiten la interacción con los lectores mediante comentarios y son una herramienta popular para compartir conocimientos, promocionar productos o simplemente expresar ideas en internet.
_REFERENCIA_ Blog web

Web store

Una web store o tienda en línea es una plataforma digital donde se venden productos o servicios a través de internet. Permite a los usuarios navegar por catálogos, hacer pedidos y realizar pagos desde cualquier lugar. Es una forma conveniente y accesible de comprar y vender, disponible las 24 horas del día.
_REFERENCIA_ web store

Sistema de gestion de contenidos (CMS)

Un Sistema de Gestión de Contenidos (CMS) es una plataforma que permite crear, editar y administrar contenido digital en un sitio web sin necesidad de conocimientos avanzados de programación. Facilita la organización de textos, imágenes, videos y otros elementos, siendo ideal para blogs, tiendas en línea o páginas corporativas. Ejemplos populares de CMS son WordPress, Joomla y Drupal.
_REFERENCIA_ CMS
